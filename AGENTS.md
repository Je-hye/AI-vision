# 에이전트 지침 (AGENTS.md)

이 파일은 이 프로젝트에서 작업하는 AI 에이전트를 위한 지침을 담고 있습니다.

## 일반 원칙
- 모든 답변과 문서는 **한국어**로 작성합니다.
- 코드 변경 사항은 항상 테스트를 통해 검증합니다.
- 새로운 기능이나 버그 수정 시 `WORK_LOG.md`를 업데이트합니다.
- 향후 계획은 `ROADMAP.md`에 기록합니다.

## 기술 스택
- **Backend**: FastAPI, OpenCV, Pydantic
- **Frontend**: HTML/CSS/JS (Vanilla)
- **AI**: Roboflow API
- **Testing**: Pytest
