# 향후 로드맵 (ROADMAP.md)

## 단기 목표 (1-2주)
1.  **UI/UX 개선**:
    *   대시보드 디자인 고도화 (보다 세련된 인터페이스).
    *   실시간 그래프 또는 통계 시각화 추가.
2.  **안정성 강화**:
    *   스트림 연결 끊김 시 자동 재연결 로직 구현.
    *   에러 발생 시 사용자 알림 기능 강화.
3.  **한국어 문서화**:
    *   모든 소스 코드 내 주석 및 기존 문서를 한국어로 번역/보강.

## 중기 목표 (1개월)
1.  **성능 최적화**:
    *   스트림 처리 프로세스의 오버헤드 감소.
    *   비동기 처리 효율성 개선.
2.  **기능 확장**:
    *   로컬 추론(Local Inference) 옵션 추가 (TFLite, ONNX 등).
    *   다양한 모델 지원 및 설정 UI 제공.

## 장기 목표
1.  **모바일 최적화**:
    *   모바일 브라우저에서도 원활하게 작동하도록 반응형 설계 강화.
2.  **배포 자동화**:
    *   Docker를 사용한 간편한 배포 환경 구축.
